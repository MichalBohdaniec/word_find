Find is a command tool that outputs a string chosen by the user. 
Input text can be given in form of .txt file or as a string.
Results can be outputed in form of .txt file or as a string.

The syntax for find tool : 
	find <prompt string> -i <input.txt> -o <output.txt> <optional -c>

NOTE : EXECUTABLE ONLY WITH PARAMETERS SPECIFIED.
